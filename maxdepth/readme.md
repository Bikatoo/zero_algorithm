# 二叉树的最大深度


##### 给定一个二叉树，找出其最大深度。二叉树的深度为根节点到最远叶子节点的最长路径上的节点数。     

**示例1**
> 给定二叉树 [3,9,20,null,null,15,7]   
```
   3
  / \
 9  20
   /  \
  15   7
```
> 返回它的最大深度 3 。

**思路1**
> dfs     
> 最大深度为子树的最大深度+1     
> 即可递归计算子树的最大深度，出口为子节点为空    

**实现**
> 见 MaxDepth 源码
> 时间复杂度 O(n) 空间复杂度 O(n)   
> n取决于树的高度
